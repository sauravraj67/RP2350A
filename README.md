# RP2350A

I made a custom RP2350A devboard  , it has a USB type C and i tried to keep the size of the board as small as possible and to have as much as useful pins available on the board..

## Overview
This project is a **compact 2-layer development board** based on the **Rp2350A**. It exposes **all usable GPIOs**, includes onboard **3.3V regulation**, and USB connectivity for easy programming and prototyping. 

## Features
- RP2350A Chip 
- 2-layer compact PCB
- USB connection for programming and power
- Onboard 3.3V voltage regulation
